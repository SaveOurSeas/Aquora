Aquora is an autonomous water monitoring and exchange robot system designed to measure water quality parameters such as pH and turbidity, then coordinate with a second robot (Aqua) to transfer or exchange water based on the collected data.

🔍 Overview

Aquora operates as a smart environmental monitoring unit. It navigates to water sources, analyzes water quality in real time, and communicates results to a secondary robot system for further action. This enables automated decision-making in water quality management systems.

⚙️ Key Functions
Measures water pH levels
Measures turbidity (water clarity)
Navigates to designated water sources
Sends collected data to the Aqua robot
Triggers or coordinates water transfer actions
Displays live sensor readings
🧠 System Workflow
Aquora moves to a water source
Sensors collect pH and turbidity data
Data is processed onboard
Results are transmitted to Aqua robot
Aqua performs water exchange/handling tasks
Values are displayed for monitoring and analysis
🔬 Components Used (example — adjust to your real build)
Microcontroller (Arduino / ESP32 / Raspberry Pi)
pH sensor
Turbidity sensor
Motor driver system
DC motors / chassis
Wireless communication module (Bluetooth / WiFi / RF)
Display module (LCD / OLED)
📊 Output
Real-time pH readings
Turbidity level readings
System status updates
Communication logs between robots
🧩 Communication

Aquora communicates with the Aqua robot using a wireless protocol (e.g., Bluetooth, WiFi, or RF), sending structured sensor data for decision-making.
